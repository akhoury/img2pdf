img2pdf
=======

images to pdf util built on top of pdfkit

You need Node.js to be installed, google it

## Usage: ##
    git clone https://github.com/akhoury/img2pdf.git
    cd img2pdf
    npm install
    cd bin
    node i2p.js -h // and read
    // example
    node i2p.js --input [path] --output[path] -m --scale 0.5
